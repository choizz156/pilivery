=== 회원가입

[[validation]]
==== Validation
|===
| path | description

| email
| `이메일 형식을 지켜야합니다. ex)test@gmail.com`

| password
| `특수문자와 숫자를 포함한 8자리 이상이어야 합니다.`

| realName
| `공백이 있어서는 안됩니다.`

| phone
| `'010-xxxx-xxxx' 형식이어야 합니다.`
|===
